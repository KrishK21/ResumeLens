## Project Overview

ResumeForge is a full stack machine learning application that helps job seekers optimize their resumes for specific job postings while preserving the accuracy of their professional experience.

Instead of relying on a third-party AI service, ResumeForge uses a custom-trained machine learning model to analyze job descriptions, identify the most valuable skills and keywords, and intelligently rewrite existing resume bullet points using only the user's real work experience.

The model is designed to improve Applicant Tracking System (ATS) compatibility by learning relationships between job requirements and resume language. It enhances existing bullet points without creating fake experience, inventing projects, or exaggerating accomplishments.

The application compares a user's uploaded resume against a target job description, generates optimized bullet point suggestions, allows the user to review or edit each recommendation, and produces a new ATS-friendly, machine-readable PDF while preserving the original resume layout.
